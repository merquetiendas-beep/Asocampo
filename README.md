# ASOCAMPO 🌱

**Asociación de Mujeres Campesinas de Córdoba - Quindío**

Plataforma digital oficial para la comercialización de productos agrícolas frescos, artesanías y alimentos procesados de nuestra región.

## 🚀 Funcionalidades
- Catálogo de productos con precios finales (incluye comisión ASOCAMPO del 10%)
- Registro de productores (hasta 5 productos por usuario)
- Búsqueda y filtrado de productos
- Panel de administración **seguro** (acceso solo con contraseña)
- Conexión directa por WhatsApp para pedidos

## 📂 Estructura del proyecto
- `index.html` → Página principal y tienda virtual
- `registro.html` → Registro de nuevos productores y proveedores
- `admin.html` → Panel de control exclusivo para administrador
- `imagenes_productos/` → Carpeta con todas las imágenes de productos
- `contactos_asocampo.xlsx` → Base de datos y contabilidad interna
